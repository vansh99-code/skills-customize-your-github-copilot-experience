
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman-style word guessing game to practice Python strings, loops, conditionals, user input, and game state management.

## 📝 Tasks

### 🛠️ Create the Hangman Game Logic

#### Description

Write a Python program that selects a random hidden word and allows the player to guess letters one by one.

#### Requirements
Completed program should:

- Choose a hidden word from a predefined list of words.
- Show the current word progress using underscores for unknown letters (e.g. `_ a _ g m a n`).
- Allow the player to enter one letter per guess.
- Reveal matching letters in the hidden word when guessed correctly.
- Track and display the number of incorrect guesses remaining.

### 🛠️ Add Game Flow and End Conditions

#### Description

Enhance the game by handling win/lose conditions and giving feedback after each guess.

#### Requirements
Completed program should:

- End the game with a win message when the player guesses the full word.
- End the game with a lose message when the player runs out of allowed incorrect guesses.
- Display the correct word if the player loses.
- Show current guessed letters and remaining attempts after each guess.
