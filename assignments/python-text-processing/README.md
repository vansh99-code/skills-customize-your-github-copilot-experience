# 📘 Assignment: Python Text Processing

## 🎯 Objective

Practice string manipulation and file I/O in Python by reading text files, analyzing content, and writing transformed output.

## 📝 Tasks

### 🛠️ Load and inspect text data

#### Description
Read a text file and report basic information about its contents.

#### Requirements
Completed program should:

- Open a text file using Python file I/O.
- Read the file contents into a variable.
- Count the number of words in the text.
- Count how many times a specific word appears.
- Print the results clearly.

### 🛠️ Search and transform text

#### Description
Find sentences containing a keyword and apply text transformations.

#### Requirements
Completed program should:

- Split the text into sentences.
- Find all sentences that contain a target keyword.
- Convert those sentences to uppercase or title case.
- Print the transformed sentences.

### 🛠️ Save results to a new file

#### Description
Write a summary file that captures the analysis and transformed text.

#### Requirements
Completed program should:

- Create a summary string with word counts and keyword results.
- Write the summary to a new output file.
- Include at least one transformed sentence in the saved file.
- Confirm the output file was written successfully.
